Это программа -  клиент-серверный калькулятор.

Клиент забивает в поля адрес сервера, порт выражение, и нажимает на кнопку "=", или же клавишу enter. Звук, вибро отключаются. Из истории примеры можно удалять "смахиванием" в сторону.

Сервер поддерживает множественные подключения. Отслеживает соединения с сетью. Для запуска нужно указать порт, или как в случае с клиентом, можно ничего не указывать оставив значения по умолчанию.
Пакет expr  в коде серевер - это библиотека с .jar версией которой программа выбрасывала исключение и не работала, поэтому я её использовал как пакет.
